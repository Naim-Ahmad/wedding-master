###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: ?

<i>Write your explanation here</i>
- C: 3
In the array, there is a 0, which is a falsy value. So, even though the length of the array is 4, the iteration will occur 3 times, and each time, the value of 'count' increases by one. The result is 3.
</p>
</details>
